# AI National Testing Service (AINTS)

AI National Testing Service (AINTS) is an AI-powered online examination platform designed to conduct secure, scalable, and automated exams with real-time monitoring and cheating detection.

It combines web technologies with AI-based proctoring to improve fairness and integrity in online assessments.

---

## 🚀 Features

- AI-based exam monitoring (cheating detection)
- Real-time WebSocket proctoring system
- Student and admin dashboards
- Test creation and management system
- Automated result evaluation
- Secure authentication system (JWT)
- File upload support for test questions
- MongoDB Atlas cloud database integration
- Live exam session tracking

---

## 🛠 Tech Stack

**Frontend:**
- React / Vite
- HTML, CSS, JavaScript

**Backend:**
- Node.js
- Express.js
- WebSocket (real-time communication)

**Database:**
- MongoDB Atlas
- Mongoose ODM

**Other Tools:**
- JWT Authentication
- Multer (file uploads)
- AI-based detection scripts
